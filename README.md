# PJ-Personalization-Of-Fashion-Style
[2024.08 ~ 2024.11] 패션 이미지 스타일 분류 프로젝트      
: 사용자 스타일 기반 상품 탐색을 위한 이미지 분류 체계 구축

## 프로젝트 배경
- 패션 플랫폼 사용자의 직접 느낀 니즈에서 출발
- 사용자는 원하는 스타일을 정확히 찾기 어려워함
- AI가 분류하지만 원본 데이터 자체의 품질 문제 존재
- **검색어-이미지 매칭 정확도 향상을 위한 메타데이터 정제 필요**

## 주요 성과
1. **데이터 품질 검증**: 파일명 분석으로 성별/스타일 통계표 자동 생성
2. **전처리 파이프라인**: 이미지 배경 제거, 배경 색상 대체, 증강 방법론 비교 후 최적 방식 도출
3. **반복 실험**: 17차 시행착오를 통한 학습
4. **핵심 인사이트**: 모델 성능보다 클래스 분류 품질 향상 우선 필요

## 실행 환경
- Google Colab (GPU: A100)
- Python 3.10
- PyTorch, torchvision, rembg

## 데이터 출처
- [AI Hub] 연도별 패션 스타일 이미지 및 추천 데이터

## 파일 구조
- `01_labeling.ipynb`: 데이터 분포 및 라벨 구조 진단
- `02_modeling.ipynb`: 모델 학습 및 검증
